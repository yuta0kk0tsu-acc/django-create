Проект: Многостраничный лендинг для лимитированной ювелирной коллекции «Dark Skies»
Цель: Презентация коллекции, стимулирование продаж через акцию, предоставление контактной информации.
Основа: Доработка и расширение исходного макета (ТЗ v1.0).

1. Общая структура и навигация
Сайт состоит из последовательных экранных блоков (секций), объединенных в единую вертикальную прокрутку.

Глобальный хедер (фиксированный или статичный):

Логотип / Название: «Dark Skies».

Главное навигационное меню: Горизонтальный ряд кнопок-табов. Элементы:

Лимитированная коллекция (возможно, активно по умолчанию)

Акция дня

Контакты

Категориальная навигация внутри каталога: Под основным меню, при отображении секции каталога, появляется второй ряд вкладок для фильтрации товаров:

Кольца | Серьги | Кулоны | Печатки

Поведение: Клик по вкладке отображает соответствующие товары. Возможно, реализация через anchor-ссылки или JavaScript-фильтрацию.

2. Контентные секции (по порядку следования)
Секция 1: Каталог товаров (со стартовым фильтром "Кольца")

Заголовок: «Лимитированная коллекция» (отображается при активации соответствующей вкладки в главном меню).

Список товаров в выбранной категории, представленный в виде карточек. Формат карточки:

Тип товара (например, Кольцо)

Название модели в кавычках (например, "Жемчужный сон")

Цена (формат: 12,000 ₽ — с запятой как разделитель тысяч).

Состояние по умолчанию: При загрузке страницы или клике на «Лимитированную коллекцию» должна отображаться категория «Кольца».

Секция 2: Расширенное описание продукта

Формат: Текстовая блокировка с атмосферным описанием.

Контент: Маркетинговый текст, посвященный конкретной модели/набору (например, кольцам «Поцелуй времени»). Содержит заголовок, поэтическое описание для общей концепции и отдельные абзацы, характеризующие мужскую и женскую версии.

Примечание: Судя по скриншотам, этот блок может быть частью секции каталога (например, при выборе конкретного товара) или самостоятельной секцией.

Секция 3: Акционный блок («Акция дня»)

Заголовок: «Акция дня».

Контент:

Основное акционное предложение:
Название: Кольцо "Корень"
Цена: 7.000 ₽ (крупным шрифтом)
Старая цена: -10.000 ₽ (перечеркнута, для сравнения).
Дополнительная информация: Текст, который, судя по расположению, может относиться к предыдущему описательному блоку (о кольцах «Поцелуй времени»), включая итоговую цену (16.000 ₽).

3. Функциональные требования
Навигация: Клик по пунктам главного меню плавно прокручивает страницу к соответствующей секции. Пункт меню визуально подсвечивается, когда его секция находится в области просмотра.

Фильтрация каталога: Клик по вкладке категорий (Кольца, Серьги и т.д.) динамически отображает только товары из выбранной категории. Рекомендуется реализация без перезагрузки страницы (JavaScript).

Адаптивность: Макет должен быть отзывчивым (responsive) и корректно отображаться на мобильных устройствах. Навигационные табы на мобильных могут преобразовываться в выпадающее меню или вертикальный список.

4. Технические детали и контент
Формат цен: Использовать запятую в качестве разделителя тысяч (12,000 ₽).

Язык: Русский.

Ключевые элементы для вёрстки: Особое внимание уделить стилизации акционного блока (контрастные цвета, перечёркнутая цена) и кликабельным контактным данным.
